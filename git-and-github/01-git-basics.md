# Git Basics for DevOps

Git is a distributed version control system used to track code, infrastructure,
and configuration changes.

## Common Commands
- git init
- git clone <repo>
- git status
- git add .
- git commit -m "message"
- git push
- git pull

## Git Areas
- Working Directory
- Staging Area
- Repository

## Check History
- git log
- git log --oneline

## DevOps Use Case
Used to manage application code, CI/CD pipelines, Terraform files,
Kubernetes manifests, and scripts.
